# Mission

## 個人開発のタスク管理を楽に。

# Vision

個人開発では、タスク管理がめんどくなってくる。
しかしAI時代において適切なタスク、スコープ管理は大切。

issue-cli は、タスク管理をローカルに保存し、
AI と人間が同じコンテキストを共有しながら開発できる状態を作る。

# Product Principles

## 1. Local-first

ローカル環境だけでも完結して利用できる。

## 2. AI-readable

ドキュメントは AI Agent が読める構造にする。

## 3. Simple over Enterprise

複雑な組織向け機能より、
個人開発者の使いやすさを優先する。

## 4. Markdown-native

可能な限り Markdown を source of truth にする。
